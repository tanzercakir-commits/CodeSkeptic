MCP server (--serve): stdio JSON-RPC 2.0 вҖ” initialize / tools/list /
tools/call(analyze). Agents call it after every edit; findings return
as structured JSON with dataflow traces.

Data flow:
  SummaryRegistry.rebuild(TU) в”Җв”Җfirstв”Җв”Җв–¶ Rule.check()
  Rule.check() в”Җв”ҖrunDataflowв”Җв”Җв–¶ DataflowEngine
    Phase 1: fixpoint (transfer only вҖ” PURE)
    Phase 2: reporting pass (onStatement вҖ” with fixpoint states)
  Diagnostic(+TraceNotes) в”Җв”Җв–¶ canonical path вҶ’ suppression вҶ’ baseline
    вҶ’ severity filter вҶ’ sort+dedup в”Җв”Җв–¶ Reporter

Engine contracts (kept in sync with todo.md "Technical Notes"):
  - transfer is pure; reporting happens only in the post-fixpoint pass
  - CFG is fine-grained (setAllAlwaysAdd): analyses look only at the
    top node of each element; CFGs come from the shared CfgCache
    (one build per function)
  - assume edges: succ[0]=true, succ[1]=false; refineOnEdge is called
    only on two-successor terminators
  - the iteration cap scales with lattice height (latticeHeight hook)
  - summary/CFG keys are TU-local FunctionDecl* вҖ” cleared at TU end

Diagnostic (data shape):
  severity, file (canonical), line, column, rule_id, message,
  notes[] = dataflow trace steps (file, line, column, message)
