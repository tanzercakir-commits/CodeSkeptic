# Engineering journal (devlog)

The unabridged, chronological record behind the curated
[ROADMAP.md](../../ROADMAP.md):

- **[ROADMAP-full.md](ROADMAP-full.md)** — assessment & roadmap as a
  living document: every scan campaign (systemd, shadPS4, libgit2,
  llama.cpp, Godot, Carbon, JoltPhysics, TFLite, …), every
  false-positive family and the engine feature it became, the thesis
  test and the recall pivot, ablations and honest negatives.
- **[changelog.md](changelog.md)** — PR-by-PR change history with
  rationale and receipts.

These files are deliberately kept out of the repository root: they are
long by design (the project's evidence trail), and the root is for
first-time readers. Nothing here is prettified for an audience — that
is what makes it useful.
